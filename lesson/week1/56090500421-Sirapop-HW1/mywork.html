Project

The topic for my internship project is "Application of Machine Learning to the Prediction of Ames Mutagenicity". The benefit of this project is to reduce time used for Ames test research for drug design. I need to do some experiments on two approaches of classification which are numerical approach and text mining approach.

Two approaches
For numerical approach, I needed to use PADEL-Descriptor to calculate numerical vectors from SMILES format compounds in the mutagenicity benchmark dataset, write R code to load the file gotten from the calculation and train Kernlab Support Vector Machine with it, evaluate the machine with 10-fold cross validation and take note on the results. After this process was done, I needed to improve the performance of KSVM with some feature selection algorithms which for this case, I chose to implement Random Forest algorithm and Information Gain technique to select only important features for the training. As a result, the training took lesser time but the performance were not improved since there seemed to be lesser accuracy.
For text mining approach, I needed to learn how to use ConText, a Convultional Neural Network for text classification. The data preparation was really hard since the software package would only process with text data that is separated into sentences of words. Each word inside a sentence needs to be separated by a space. Thus, I needed to write some Java programs to help me. However, it was not so easy as it sounded. The text data that I needed to use for the classification is in SMILES format so I also needed to understand it a bit.
