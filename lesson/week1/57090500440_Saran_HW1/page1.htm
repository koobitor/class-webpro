The basic rule is, "On a sunny day set aperture to f/16 and shutter speed to the [reciprocal of the] ISO film speed [or ISO setting] for a subject in direct sunlight."

Sunny 16 Rule

           In photography, the sunny 16 rule (also known as the sunny f/16 rule) is a method of estimating correct daylight exposures without a light meter. (For lunar photography there is a similar rule known as the looney 11 rule.) Apart from the obvious advantage of independence from a light meter, the sunny 16 rule can also aid in achieving correct exposure of difficult subjects. As the rule is based on incident light, rather than reflected light as with most camera light meters, very bright or very dark subjects are compensated for. The rule serves as a mnemonic for the camera settings obtained on a sunny day using the exposure value (EV) system.

An elaborated form of the sunny 16 rule is to set shutter speed nearest to the reciprocal of the ISO film speed / setting and f-number according to this table

 Table Sunny 16 Rule
Lighting conditions Aperture ISO* Shutter Speed* Shadow detail
Snow/sand f/22 100 1/125 Dark with sharp edges
Sunny f/16 100 1/125 Distinct
Slight overcast f/11 100 1/125 Soft around edges
Overcast f/8 100 1/125 Barely visible
Heavy overcast f/5.6 100 1/125 No shadows
Open shade/sunset f/4 100 1/125 No shadows
Backlighting Add one stop 100 1/125 n/a 
*You can set ISO, but you'll change Shutter Speed nearby 1/ISO too. 
Ex : ISO = 200, you'll set Shutter Speed = 1/250(nearby ISO).
      ISO = 400, you'll set Shutter Speed = 1/500(nearby ISO).
